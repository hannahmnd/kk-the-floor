# Kultuuriklubi THE FLOOR timer
autor: Hannah Mänd
loodud GAMES NIGHT 17.03.2026 jaoks

## Klaviatuurikäsud

| Klahv | Tegevus |
|-------|---------|
| **F** | Mängija 1 - õige vastus |
| **S** | Mängija 1 - skip |
| **J** | Mängija 2 - õige vastus |
| **L** | Mängija 2 - skip |
| **SPACE** | Start / Stop |
| **N** | Uus mäng (pärast mängu lõppu) |
| **B** | Tagasi seadistusekraanile |
| **SHIFT+F** | Täisekraani sisse/välja |
