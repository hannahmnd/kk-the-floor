# Kultuuriklubile THE FLOOR timer
autor: Hannah Mänd
esimene kasutus: 17.03.2026

## Klaviatuurikäsud

| Klahv | Tegevus |
|-------|---------|
| **F** | Mängija 1 - õige vastus |
| **S** | Mängija 1 - vahele |
| **J** | Mängija 2 - õige vastus |
| **L** | Mängija 2 - vahele |
| **SPACE** | Start / Stop |
| **N** | Uus mäng (pärast mängu lõppu) |
| **B** | Tagasi seadistusekraanile |
| **SHIFT+F** | Täisekraani sisse/välja |
